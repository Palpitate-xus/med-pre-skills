# Sub-Skill: 完整示例 — RRMS 复发预测

> **所属 Meta-Skill:** [临床预测模型开发指南](../clinical-prediction-model-development-zh.md)

---

## 研究背景

- **疾病：** 复发缓解型多发性硬化症（Relapsing-Remitting Multiple Sclerosis, RRMS）
- **临床问题：** 哪些患者在未来 2 年内有较高的复发风险？
- **临床决策：** 是否从一线疾病修正治疗（DMT）升级到高效 DMT？

---

## 13 步应用

### Step 1: 目的与使用者
- **目标人群：** 已确诊 RRMS、正在接受一线 DMT 的患者
- **使用者：** 神经内科主治医生（门诊随访场景）
- **决策：** 预测 2 年复发概率，指导治疗升级

### Step 2: 现有模型检索
- 检索发现若干 RRMS 复发预测模型
- 但多数未在中国人群中验证，或方法学质量不高（PROBAST 评价）
- **决定：** 开发新模型，计划后续在中国人群中外部验证

### Step 3: 数据来源
- 多中心回顾性队列（5 个神经科中心）
- 纳入 2018-2022 年间新诊断 RRMS 患者
- EHR 提取 + 人工复核复发记录

### Step 4: 结局定义
- **主要结局：** 2 年内是否出现临床复发（二分类）
- **复发定义：** 新发神经功能缺损持续 ≥24 小时，排除发热/感染
- **确认：** 两名神经科医生独立判定

### Step 5: 预测变量
| 变量 | 类型 | 理由 |
|------|------|------|
| 年龄 | 连续 | 文献一致关联 |
| 病程 | 连续 | 疾病活动性指标 |
| 基线 EDSS | 连续（保留连续） | 残疾程度 |
| 既往复发次数 | 连续 | 疾病活动性 |
| MRI 增强病灶数 | 连续 | 炎症活动 |
| 首发症状类型 | 分类 | 预后因素 |
| DMT 类型 | 分类 | 治疗强度 |

**未分类任何连续变量。**

### Step 6: 样本量
```python
# Python 手动计算（见 Sub-Skill 06）
from sub_skills_06 import pmsampsize_binary

n = pmsampsize_binary(
    parameters=15,
    prevalence=0.30,
    epv=20

# 结果：需要约 1000 例，实际可用 920 例（略不足，需在讨论中说明）
```

### Step 7: 缺失数据
- 缺失率：EDSS 5%，MRI 病灶 12%，其余 < 3%
- 方法：`miceforest` MICE，10 次插补
- 辅助变量：MRI 总病灶数、脑脊液指标
- 考虑中心聚类效应

### Step 8: 建模方法
比较三种方法：
1. Logistic 回归 + 限制性立方样条（`statsmodels + patsy`）
2. GAM（`pygam`）
3. 岭回归 Logistic（`sklearn LogisticRegression(penalty='l2')`）

### Step 9: 模型拟合
- 在每个插补数据集分别拟合
- 预测值取平均（策略 B）
- 比较三种方法的预测一致性

### Step 10: 区分度
| 方法 | 表观 AUC | 校正 AUC | 95% CI |
|------|----------|----------|--------|
| Logistic + 样条 | 0.82 | 0.78 | 0.74-0.82 |
| GAM | 0.83 | 0.77 | 0.73-0.81 |
| 岭回归 | 0.80 | 0.77 | 0.73-0.81 |

### Step 11: 校准度
- **校准截距：** -0.03（接近 0，无系统偏移）
- **校准斜率：** 0.92（略 < 1，轻微过拟合）
- **校准图：** 在预测概率 < 35% 范围内校准良好；> 35% 区域 CI 变宽（事件少）

### Step 12: 临床效用（DCA）
- **临床阈值范围：** 医生通常在 15%-30% 阈值考虑升级治疗
- **DCA 结果：** 模型在此范围内净获益高于 "全升级" 和 "全不升级"
- **解读：** 使用模型可避免不必要的治疗升级（减少副作用和费用）

### Step 13: 验证与报告
- **Bootstrap：** 500 次， optimism-corrected 性能
- **Internal-external CV：** 留一中心验证，AUC 0.73-0.82（中心间异质性可接受）
- **TRIPOD：** 完整报告
- **代码：** GitHub 公开 + Streamlit App 在线计算器

---

## 关键发现

1. **最优模型：** Logistic + 限制性立方样条（平衡了性能和可解释性）
2. **校准：** 在常见风险范围内可靠
3. **临床价值：** 在标准决策阈值范围内优于默认策略
4. **局限：** 高概率区域校准不确定，需要更大样本或更长期随访

---

## 学习要点

- 即使 AUC 相近，Logistic + 样条的可解释性优于纯 ML 方法
- 校准图揭示的信息比单一 AUC 值更丰富
- DCA 直接回答了 "这个模型在临床上有用吗"
- Internal-external CV 提前发现了中心间异质性，提示外部验证的重要性
