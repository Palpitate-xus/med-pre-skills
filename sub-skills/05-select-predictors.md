# Sub-Skill 05: 基于先验知识选择预测变量

## 核心任务

选择一组在基线（time zero）可获取、有生物学/临床合理性、且在目标场景中可测量的预测变量。

---

## 选择原则

### 1. 基于先验知识，而非纯数据驱动

| 方法 | 是否推荐 | 原因 |
|------|----------|------|
| 文献综述 + 专家共识 | ★★★ | 基于证据和机制 |
| 临床指南推荐变量 | ★★★ | 已有共识 |
| 生物学通路分析 | ★★★ | 因果机制支撑 |
| 单变量 P 值筛选（P < 0.05） | ☗ | 增加偏倚，导致过拟合 |
| 逐步回归 | ☗ | 同上单变量问题 |
| 纯机器学习的特征选择 | ★★☆ | 仅在大样本 + 有验证时可行 |

### 2. 连续变量保持连续

**绝不要对连续变量进行任意分类：**
- 年龄分 "老年/非老年" → 丢失年龄与结局的真实关系
- BMI 分 "正常/超重/肥胖" → 人为制造阈值跳跃

**正确做法:** 使用样条函数（splines）或 GAM 捕捉非线性关系

### 3. 考虑最终使用者

```
这个变量在目标临床场景中:
├── 是否常规测量？
│   └── 否 → 考虑排除（除非极其重要）
├── 测量成本如何？
│   └── 昂贵/侵入性 → 需权衡增量价值
├── 测量标准化程度？
│   └── 不同中心/操作者差异大 → 可能不稳定
└── 时间一致性？
    └── 是否总在基线前完成？
```

---

## 预测变量分类

| 类型 | 示例 | 注意事项 |
|------|------|----------|
| 人口学 | 年龄、性别、种族 | 年龄保持连续 |
| 病史 | 合并症、既往事件 | 使用标准化评分（如 Charlson） |
| 临床表现 | 症状严重程度、体征 | 使用经验证的量表 |
| 实验室 | 血常规、生化、生物标志物 | 注意参考范围差异 |
| 影像学 | CT/MRI 特征 | 考虑影像组学（需大样本） |
| 治疗 | 当前用药、手术史 | 治疗变量可能与结局有复杂关系 |

---

## 关于交互项

- **一般不建议**在初始模型中大量纳入交互项
- 仅在**有强先验假设**时考虑（如某药物仅在特定年龄组有效）
- 交互项消耗大量自由度 → 需要更大样本量

---

## 质量检查清单

- [ ] 预测变量选择基于证据，而非纯数据驱动
- [ ] 连续型预测变量保持连续（使用样条）
- [ ] 所有预测变量在基线（time zero）即可获取
- [ ] 预测变量在目标临床场景中现实可测
- [ ] 预测变量集合与先验文献/指南一致
- [ ] 已记录排除某些变量的理由
